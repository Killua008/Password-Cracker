# Password-Cracker
Password-Cracker is a Python based project.
In this Project we are using two methods to crack a password .
For first method we have Common Password which checks the uploaded common password.txt file which contains 10k common passwords.
If your password comes under this file then its and common password and provide you the time and attempts it took .
For second method we have Brute Force which is used when the password in not in the File. The result solely depends on the processor' speed.
In the end result after cracking the password it will provide you the time and attempts it took .
The program provides a simple GUI for user friendly purpose.
